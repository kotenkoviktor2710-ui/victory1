# README — сюжет визуальной новеллы «Последний День Курьера»

## Концепт

**Жанр:** визуальная новелла, городской детектив, тайм-парадокс, поиск предметов.  
**Формат:** повторяющийся день, 5 основных веток, 2 скрытые ветки, несколько финалов.  
**Тон:** загадочный, динамичный, без тяжёлого хоррора, подходит для широкой аудитории.

Главный герой, курьер **Артём Воронов**, просыпается утром и находит у себя дома странную посылку без номера заказа. В течение дня он получает сообщения от системы **«Синхрон»**, встречает людей, связанных с городским экспериментом, и в 19:43 становится свидетелем события на центральном перекрёстке. После этого день начинается заново.

Постепенно игрок узнаёт: **«Синхрон»** — городская система прогнозирования, которая должна предотвращать аварии, пробки и чрезвычайные ситуации. Но система застряла в логической ошибке: она видит, что в 19:43 случится опасное событие, но не может найти безопасный сценарий. Поэтому она повторяет день, меняя маршруты людей.

Главная мысль: **идеальный день нельзя построить только алгоритмом. Люди ошибаются, но именно люди способны сделать выбор, который невозможно просчитать.**

## Переменные для ветвлений

```text
trust_max          — доверие Макса
trust_lera         — доверие Леры
trust_orlova       — доверие Орловой
trust_nika         — доверие Ники
sync_obedience     — насколько игрок слушается «Синхрон»
humanity_score     — насколько игрок помогает людям
parcel_focus       — насколько игрок сосредоточен на посылке
observer_score     — насколько игрок наблюдает и собирает данные
evidence_count     — количество ключевых улик
chaos_score        — насколько игрок ломает ход дня
cycle_count        — номер повторения дня
```

## Карта веток

| Код | Ветка | Условия открытия | Суть |
|---|---|---|---|
| A | Спасти незнакомку | humanity_score >= 3, выбор спасти девушку в 19:43 | Эмоциональная ветка спасения |
| B | Ошибка курьера | parcel_focus >= 3 | Расследование посылки |
| C | Город без людей | chaos_score >= 2 или humanity_score <= 1 | Плохая тревожная ветка |
| D | Союз с алгоритмом | sync_obedience >= 4 | Игрок помогает системе |
| E | Настоящий конец дня | evidence_count >= 8, trust_lera >= 3 | Главный финал |
| S1 | Память города | observer_score >= 3, evidence_count >= 4 | Скрытая ветка наблюдателя |
| S2 | Нулевой курьер | parcel_focus >= 4, несколько доставок посылки | Скрытая ветка прошлого курьера |

# Как сделать игру бесконечной после финала

После прохождения настоящего финала можно открыть режим **«Новые маршруты»**.

В нём «Синхрон» уже не перезапускает день, но иногда отправляет Артёму странные необязательные заказы. Каждый заказ — отдельная мини-история:

- пропавший подарок;
- забытый дневник;
- письмо без адресата;
- потерянная собака;
- заказ из старого цикла;
- посылка от Нулевого курьера;
- странный маршрут без получателя.

Так можно постоянно добавлять новые эпизоды без переписывания основной истории.
