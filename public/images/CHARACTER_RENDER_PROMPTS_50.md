# CHARACTER_RENDER_PROMPTS_50.md

Файл содержит 50 отдельных промптов для генерации персонажей игры. У каждого персонажа свой промпт, без общего master prompt.

Рекомендуемый формат генерации: один персонаж за раз, PNG с прозрачным фоном, полный рост, cartoon 3D, игровая модель для merge/clicker/PvP.

## 01. Threadpaw

**Тип:** синий плюшевый кот  
**Игровая роль:** быстрый атакующий  
**Палитра:** синий, фиолетовый, жёлтые пуговицы  
**Ключевые детали:** широкая зубастая улыбка, длинный хвост, крупные лапы

### Prompt

```text
Create a single unique collectible toy monster character named "Threadpaw" for a cartoon-style mobile merge battler game.

Character concept:
Threadpaw is a синий плюшевый кот designed as a factory-made mascot toy. The character must feel cute, strange, slightly creepy, but still appealing and suitable for ages 8+.

Visual design:
Use синий, фиолетовый, жёлтые пуговицы as the main color palette. Add широкая зубастая улыбка, длинный хвост, крупные лапы. The character has a large expressive head, small compact body, short legs, oversized hands and feet, strong readable silhouette, premium plush or toy-like materials, visible stitching, clean fabric seams, button eyes, patchwork details, soft fur or polished plastic parts where appropriate.

Personality:
The character should visually communicate the role of быстрый атакующий. Make the pose dynamic and game-ready, with clear emotion and readable body language.

Rendering:
High-quality cartoon 3D render, AAA mobile game asset quality, polished toy factory mascot design, clean shapes, detailed materials, soft studio lighting, ultra sharp, full body, centered composition, isolated character, no background, transparent background, no text, no logo, no watermark.
```
